Design System

Calm by design.
Light is the only accent.

The visual language behind LumenUI — a focused UI suite for World of Warcraft. Roughly 98% monochrome, 2% accent: a graduated near-black canvas where brightness itself signals what's active. Pick an accent above and watch the whole system re-tint.

The suite shell

The settings dashboard lives its own modern identity — the world documented on this page. Quiet surfaces, one clean sans, light as the accent.

The gameplay layer

In-combat frames

Raid frames, unit frames and nameplates stay close to the WoW original — class colours, familiar shapes. Under pressure, recognition beats style. This page does not govern them.

01

Colour

Components never name a colour — only a semantic role. The raw palette lives in exactly one place; everything else reads a token. The neutrals are deliberately hue-free: depth comes from the lightness step between layers, not a tint — so any chosen accent, warm or cool, sits on the canvas without a fight.

02

Accent engine

One base colour, everything else derived. Ship monochrome; let a user opt into a hue. Every state below is computed from the single base — no accent variant is ever hand-maintained. This page gathers every accent placement at once to document them; in the running shell the accent stays a small, deliberate fraction — roughly 98% monochrome, and Pure Light ships as the default.

Derivation

hover = base mixed 35% toward white · pressed = 12% toward black · selection = base at 14% · focus at 55% · glow at 35% · wash at 10%. Swapping the base recomputes all of them and re-tints the entire interface in place.

Curated presets

Pure Light is the shipping default. Only primary actions and true state (a switch, a selected row, focus) take the accent — sliders too. Body text, card fills and structural borders stay mono, always.

03

Typography

One clean sans everywhere — Inter (SIL OFL), bundled with the addon. Components name a weight, never a typeface, so the face swaps in one place. State never changes weight — glyph widths would jump; the fill carries the state instead.

04

Radius & spacing

A five-step radius scale and one spacing rhythm. Nesting rule: outer radius = inner radius + padding. Ring assets carry a 2px stroke — a single pixel vanishes at the shell's 0.8 render scale.
